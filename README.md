# SwiftKey Development Limited Website

A modern and animated React.js business website developed for **SwiftKey Development Limited**.  
This project showcases commercial kitchen solutions, hospitality interiors, restaurant fit-outs, and construction services with premium UI/UX, cinematic animations, and responsive layouts.

# ⚡ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/simran9910/swiftkey_project.git
```

## Move Into Project Folder

```bash
cd swiftkey_project
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm start
```

---

# 🖥️ Build For Production

```bash
npm run build
```
